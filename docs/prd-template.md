## 2. 제품/기능 개요

이 기능이 어떤 사용자 문제를 해결하는지 3~5문장으로 설명한다.

예시:

- 사용자가 본인과 비슷한 체형의 리뷰를 더 빠르게 찾도록 돕는다.
- 셀러가 상품별 핏 불만 원인을 확인하고 상세페이지를 개선하도록 돕는다.

## 3. 문제 정의

### 사용자 문제

- 사용자가 현재 겪는 불편은 무엇인가?
- 기존 화면/흐름에서 어디가 병목인가?
- 이 문제가 구매 전환, 반품, 탐색 시간에 어떤 영향을 주는가?

### 비즈니스/운영 문제

- 셀러 또는 운영자가 겪는 문제는 무엇인가?
- 데이터가 쌓여도 의사결정으로 연결되지 않는 지점은 어디인가?
- 개선되면 어떤 지표가 좋아지는가?
