# 🗺 User Flow: Vibe-Select Experience

본 문서는 사용자가 서비스에 진입하여 최종 구매 및 인터랙션을 수행하는 흐름을 정의합니다.

## 1. 메인 진입 및 탐색 (Discovery)
1. **Landing**: 사용자가 메인 페이지 접속.
2. **Browsing**: 스크롤을 통해 큐레이션된 브랜드 스토리 및 상품 카드 탐색.
3. **Search/Filter**: 특정 카테고리나 브랜드를 필터링하여 상품 리스트 확인.

## 2. 상품 상세 및 AI/3D 체험 (Experience)
1. **Product Detail**: 상품 클릭 시 상세 페이지 이동.
2. **3D Interaction**: 
    - 3D 뷰어 활성화 버튼 클릭.
    - 모델을 회전/확대하며 소재 및 디테일 확인.
3. **AI Virtual Try-on**:
    - '가상 착장하기' 클릭.
    - 본인의 전신 사진 업로드.
    - AI가 합성된 결과물 제시 -> 만족 시 장바구니 담기.

## 3. 라이브 쇼핑 참여 (Engagement)
1. **Enter Live**: 상단 'LIVE' 플로팅 버튼 또는 메뉴를 통해 방송 진입.
2. **Watch & Chat**: 실시간 영상 시청 및 Socket.io 기반 채팅 참여.
3. **Instant Purchase**: 방송 중 등장하는 상품 팝업 클릭 시 즉시 구매 레이어 노출.

## 4. 결제 및 완료 (Conversion)
1. **Cart**: 담아둔 상품 확인 및 수량 조절.
2. **Checkout**: 배송 정보 입력 및 결제 시뮬레이션.
3. **Success**: 주문 완료 페이지 (추천 상품 노출).

---

## 🚦 예외 상황 흐름 (Edge Cases)
- **3D 모델 로딩 실패 시**: 고해상도 2D 이미지 갤러리로 자동 전환.
- **비로그인 유저**: AI 착장이나 채팅 시도 시 로그인 모달 띄우기.
- **라이브 종료 시**: 녹화된 VOD 다시보기 모드로 전환.
