# pin-stitch 프로젝트 기획서 (v2.0)

## 1. 프로젝트 정의

`pin-stitch`는 사용자의 체형 정보와 실제 구매 리뷰를 연결해 패션 상품 구매 실패를 줄이는 맞춤형 패션 커머스 플랫폼이다.

핵심은 상품을 단순히 추천하는 것이 아니라, **나와 비슷한 체형의 사용자가 남긴 리뷰와 구매 결과를 구조화하여 구매 판단을 돕고, 셀러에게는 상품 개선 인사이트를 제공하는 것**이다.

## 2. 문제 정의

### 고객 문제

- 모델 착용 이미지와 실제 착용감 사이의 차이가 크다.
- 같은 상품이라도 키, 체형, 골격 타입, 선호 핏에 따라 만족도가 달라진다.
- 리뷰는 많지만 나와 비슷한 체형의 리뷰를 찾기 어렵다.
- 사이즈표만으로 실제 핏, 기장, 어깨, 허리 여유를 예측하기 어렵다.
- 온라인에서 본 상품을 오프라인 매장에서 입어보고 싶어도 가까운 매장 재고를 확인하기 어렵다.

### 셀러 문제

- 리뷰와 반품 사유가 쌓여도 어떤 체형의 고객이 왜 불만족했는지 파악하기 어렵다.
- 사이즈/핏 불만이 상세페이지, 옵션 구성, 상품 기획에 충분히 반영되지 않는다.
- 반복되는 핏 불만족은 반품 비용과 고객 이탈로 이어진다.

## 3. 핵심 가치 제안

> 나와 비슷한 체형의 리뷰로 더 정확하게 고르고, 셀러는 체형별 데이터를 기반으로 상품을 개선한다.

| 대상 | 제공 가치 |
| --- | --- |
| 고객 | 체형별 리뷰 요약, 체형 적합도, 사이즈/핏 주의사항, 가까운 매장 재고 확인 |
| 셀러 | 체형별 만족도, 핏 불만 키워드, 반품 사유 분석, 상세페이지/상품 개선 제안 |
| 플랫폼 | 구매 전환율 개선, 반품률 감소, 리뷰/체형 기반 데이터 자산 확보 |

## 4. MVP 범위

### 우선순위 기준

- `P0`: MVP 필수 기능
- `P1`: MVP 이후 초기 고도화
- `P2`: 장기 확장

### 고객 웹 P0

- 체형 프로필 등록/수정
- 상품 목록/검색/필터/정렬
- 상품 상세
- 체형별 리뷰 필터링
- 체형 기반 리뷰 요약
- 체형 적합도 점수 표시
- 사이즈/핏 주의사항 표시
- 샘플 매장 재고 및 위치 안내
- 장바구니/주문 기본 흐름
- 장바구니 쿠폰 자동 최적 적용

### 셀러 웹 P0

- 상품별 리뷰 분석 대시보드
- 체형별 만족도/불만 비율 조회
- 핏 키워드 분석
- 반품 사유 샘플 분석
- 상품 개선 인사이트 카드

### P1

- 반품 사유 자동 분류 고도화
- 체형별 상품 추천 고도화
- 리뷰 요약 품질 개선
- 상품 개선 가이드 자동 생성
- 실제 POS/ERP 기반 재고 연동
- 셀러 리포트 다운로드

### P2

- 사진 기반 체형 분석
- AI Virtual Try-on
- 스마트 피팅룸 요청
- 지오펜스/비콘 기반 매장 모드
- 개인 스타일링 챗봇
- 체형 기반 추천 광고 상품

## 5. 기술 스택 및 구조

## 7. 핵심 분석 흐름

1. 사용자가 키, 몸무게, 평소 사이즈, 골격 타입, 선호 핏을 등록한다.
2. 상품 리뷰를 구매자의 체형 정보 기준으로 필터링한다.
3. 사용자의 체형과 유사한 리뷰에서 핏, 사이즈, 기장, 소재 관련 문장을 추출한다.
4. 상품별 체형 적합도 점수를 계산한다.
5. 상품 상세에서 체형별 리뷰 요약과 사이즈 주의사항을 제공한다.
6. 선택한 옵션의 샘플 매장 재고를 연결해 오프라인 방문 가능성을 보여준다.
7. 셀러 대시보드에서 체형별 만족도, 부정 키워드, 반품 사유를 분석한다.

## 12. 결론

`pin-stitch` v2의 핵심은 패션 커머스의 실제 문제인 사이즈와 핏 불확실성을 리뷰 데이터로 해결하는 것이다.

MVP에서는 사용자 구매 판단과 셀러 상품 개선에 직접 기여하는 **체형 프로필, 체형별 리뷰 요약, 체형 적합도, 매장 재고 안내, 셀러 인사이트 대시보드**에 집중한다.
